# Quick Installation Guide

## 1. Install the Extension

### Chrome/Edge/Brave:
1. Download this folder to your computer
2. Open your browser and go to `chrome://extensions/`
3. Turn on **"Developer mode"** (toggle in top right)
4. Click **"Load unpacked"**
5. Select the `germa-search` folder
6. Done! 🎉

### Firefox:
1. Go to `about:debugging`
2. Click "This Firefox"
3. Click "Load Temporary Add-on"
4. Select the `manifest.json` file
5. Done! 🎉

## 2. Get Your Free API Key & Setup

1. Open a **new tab** - you should see Germa Search
2. Click **"Get Free API Key →"** to go to OpenRouter.ai
3. Sign up for free (no credit card required)
4. Get your API key from the "Keys" section
5. Return to Germa Search and enter your key
6. Click "Save & Start Searching"
7. Start searching! 

## 4. Test It Out

Try searching for:
- "machine learning research papers"
- "climate change studies" 
- "javascript tutorials"

You'll get 3 optimized search strategies for each query!

## Troubleshooting

**Extension not showing?**
- Make sure Developer mode is ON
- Refresh the extension page
- Try opening a new tab

**API key not working?**
- Make sure it starts with `sk-or-v1-`
- Check you have free credits on OpenRouter
- Try entering it again in extension options

**No search results?**
- The AI sometimes generates very specific queries
- Try clicking a different strategy (you get 3 options)
- Simplify your search terms
